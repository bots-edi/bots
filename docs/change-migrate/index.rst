Changes and Migrate
===================

* In this section details are given about changes in new versions amd also how to migrate/upgrade to newer bots versions.
* In general, try to upgrade to new bots version, especially if the version is upward compatible.
* Most of the time migrating will be easy.
* Grammars and mappings have always worked for different versions.
