Advanced Deployment
===================

We will be covering the following topics in this section:

#. If more users have access to bots-monitor, use limited rights for users.
#. If you use bots-monitor over the internet/outside your LAN, use HTTPS/SSL connection.
#. Use MySQL or PostgreSQL as the Bots database.
#. Use AS2 as a communication method.
#. Bots has options to push changes from test to production.
#. Information about performance.
